# Kothalawala Academy - Student Management System

*MTS 210603 Programming Group Assignment*

## Project Description
A simple desktop application to manage student records, marks, grades, and monthly enrollment fees for Kothalawala Academy Science classes.

## Features
- Add new students with marks and fees
- Search student by ID
- View all students with averages and grades
- Fee inventory summary
- Generate individual Report Card as CSV file
- Full GUI using Tkinter

1. Install Python 3 (if not already installed)
2. Download SMS.py

4. The GUI will open — start adding students!

## Technologies Used
- Python 3
- Tkinter (GUI)
- JSON (data storage)
- CSV (report cards)
